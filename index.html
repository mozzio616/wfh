Work from Home Project¶

wfh module¶

Setup VPN and remote desktop connection automatically.

Example

Connect to VPN and remote desktop with wake on LAN

$ python wfh.py wol
Connecting
Connected
add net 10.0.0.0: gateway ppp0
Wait a moment for startup of remote desktop.

Connect to VPN and remote desktop without wake on LAN

$ python wfh.py
Connecting
Connected
add net 10.0.0.0: gateway ppp0
Wake on LAN skipped.

class wfh.PassLogic(user_id)¶

Bases: object

Generate one-time password using PassLogic.

Parameters

user_id (str) – VPN user ID (employee ID)

Example

>>> from wfh import PassLogic
>>> pl = PassLogic('1234567')
>>> matrix = pl.get_matrix('https://passlogic')
>>> password = pl.get_matrix(matrix)
>>> print(password)
1234
